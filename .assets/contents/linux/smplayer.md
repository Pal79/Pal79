# SmPlayer telepítése

> ppa hozzáadás a csomagtárolóhoz:

```shell
sudo add-apt-repository ppa:rvm/smplayer
```

> csomagok frissítése:

```shell
sudo apt-get update
```

> telepítés:

```shell
sudo apt-get install smplayer smtube smplayer-themes
```

> eltávolítás

```shell
sudo apt-get remove --autoremove smplayer smtube smplayer-*
```
